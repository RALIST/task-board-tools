# TB-37: CodeMirror body editor in TaskDrawer

**Type:** feature
**Priority:** P1
**Size:** M
**Module:** gui
**Tags:** milestone-m3,frontend,editor
**Branch:** —
**Parent:** TB-3

## Goal

Replace read-only rendered markdown with CodeMirror editor for the body. Save → BoardService.EditTaskBody. Header + metadata block remain read-only.

## Acceptance Criteria

- [ ] CodeMirror 6 added as a dependency; editor renders the task body (everything after the metadata block) in the drawer
- [ ] Header line + first 15 metadata lines remain a read-only header above the editor (cannot be focused/edited)
- [ ] Toggle "Read / Edit" — read mode keeps the current `marked`-rendered preview; edit mode shows the CodeMirror editor
- [ ] Save → `BoardService.EditTaskBody(id, newBody)`; success toast + return to read mode; on error → toast, stay in edit mode
- [ ] Save action wired to Cmd/Ctrl+S inside the editor
- [ ] Unsaved-changes guard: drawer dismiss in edit mode prompts to discard
- [ ] After save: the drawer body reflects the new content; `BOARD.md` and the on-disk `.md` file are updated; the title line and metadata block are byte-identical to before the edit; verified end-to-end against the live `tb` binary
- [ ] Depends on TB-33
