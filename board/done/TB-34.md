# TB-34: Drag-and-drop between columns with optimistic UI and conflict revert

**Type:** feature
**Priority:** P1
**Size:** M
**Module:** gui
**Tags:** milestone-m3,frontend,dnd
**Branch:** —
**Parent:** TB-3

## Goal

Wire svelte-dnd-action into Board/Column/Card. On drop, update store optimistically and call MoveTask. On error or fsnotify-detected divergence, revert and toast.

## Acceptance Criteria

- [ ] `svelte-dnd-action` is added as a dependency and integrated into `Column.svelte`
- [ ] Dropping a card on another column updates the board store immediately (optimistic) and calls `BoardService.MoveTask`
- [ ] On `MoveTask` error → revert the optimistic update + `pushToast` the error
- [ ] If a `board:reloaded` event arrives mid-drag with the task already in a different status, the local drag is canceled and the snapshot from disk wins
- [ ] Visual affordance during drag (hover state on target column)
- [ ] Manual acceptance: drag a card backlog→in-progress; file moves on disk; `tb mv <ID> backlog` from terminal mid-drag → revert + toast
- [ ] Depends on TB-32
