# TB-35: CreateTaskDialog: modal form for new tasks

**Type:** feature
**Priority:** P1
**Size:** S
**Module:** gui
**Tags:** milestone-m3,frontend,create-dialog
**Branch:** —
**Parent:** TB-3

## Goal

Modal with title, module, type, priority, size, tags, description, optional parent and is-epic toggle. Submit calls BoardService.CreateTask.

## Acceptance Criteria

- [ ] "+" button in topbar opens a `CreateTaskDialog` modal
- [ ] Fields: title (required), module, type (default `bug`), priority (default `P2`), size (default `M`), tags (comma-separated), description, parent (optional ID dropdown from current board), is-epic toggle
- [ ] Submit → `BoardService.CreateTask`; modal closes on success; toast on error
- [ ] Esc + click-outside close the modal without submit
- [ ] New task appears in backlog within 1s of submit (via the fsnotify watcher's `board:reloaded` debounce)
- [ ] Keyboard: focus title on open; Enter submits when valid
- [ ] Depends on TB-32
