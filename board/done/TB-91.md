# TB-91: Task card is bigger then column

**Type:** improvement
**Priority:** P1
**Size:** S
**Tags:** ui,gui
**Agent:** claude
**AgentStatus:** success
**Branch:** —

## Goal

Constrain the task card so it never exceeds its column's inner width, eliminating the horizontal scrollbar that currently appears in the kanban view.

## Acceptance Criteria

- [x] Task cards in every column (backlog, in-progress, done, archive) fit within the column's content area at the default window width — no card extends past the column's right edge.
- [x] No horizontal scrollbar appears on the column list or on the page when cards are present with the longest title, longest tag, and longest module string used today on the board.
- [x] Long unbreakable strings inside a card (title, tag, module, agent label) wrap or truncate rather than forcing the card to grow horizontally.
- [x] Behavior holds when the window is resized down to the documented minimum width and when all three optional rows (meta, tags, agent badge) are present on one card.
