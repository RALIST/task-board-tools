## Goal

Extend `gui/app/preferences.go`'s `Preferences` struct (currently only `MaxWorkers`, shipped in TB-56) with three new persisted knobs — `agent_timeout_minutes`, `default_agent`, `cli_path` — plus matching Get/Set methods, range/enum validation on read, and round-trip + clamp-on-read tests.

## Context

TB-56 already laid the storage layer: JSON file at `$XDG_CONFIG_HOME/tb-gui/preferences.json`, atomic temp+rename writes, `clampMaxWorkers` warning on out-of-range values, full test matrix in `preferences_test.go`. M7 reuses that scaffold rather than introducing a second config file or a new format.

Validation rules:
- `agent_timeout_minutes` int — clamp to `[1, 240]` (1m–4h). Default `30` (matches today's `agentTimeoutDefault = 30 * time.Minute`).
- `default_agent` string — must be one of `none|claude|codex` (mirrors `tb edit -a` accepted values + the M4 dropdown). Empty / unknown → `none`.
- `cli_path` string — empty = PATH lookup (current behaviour). Non-empty values are NOT validated for existence here; `cli.NewClient` will error with `ErrBinaryNotFound` if it's broken, surfaced as a toast.

Mirror TB-56's pattern: WARN on coerce, never fail the call. JSON tags in snake_case so a future hand-edited preferences.json reads naturally.

## Acceptance Criteria

- [x] `Preferences` struct has `MaxWorkers`, `AgentTimeoutMinutes`, `DefaultAgent`, `CLIPath` (snake_case JSON tags)
- [x] Constants: `AgentTimeoutMinutesDefault = 30`, `AgentTimeoutMinutesMin = 1`, `AgentTimeoutMinutesMax = 240`; `DefaultAgentValues = []string{"none","claude","codex"}`
- [x] `SettingsService` exposes `GetAgentTimeoutMinutes() int` / `SetAgentTimeoutMinutes(int) error`, `GetDefaultAgent() string` / `SetDefaultAgent(string) error`, `GetCLIPath() string` / `SetCLIPath(string) error`
- [x] Read-time normalization: missing/zero `agent_timeout_minutes` → `AgentTimeoutMinutesDefault`; non-zero below min → min; above max → max; unknown / empty `default_agent` → `none`. All coercions emit `slog.Warn`. (Mirrors the `clampMaxWorkers` pattern in `gui/app/preferences.go:79`.)
- [x] `preferences_test.go` extended with: round-trip per field; clamp tests (timeout `0` → default, `99999` → max; default_agent `"foo"` → `none`); missing-file fallback returns defaults for all four fields
- [x] No frontend / no daemon wiring in this task — that's TB-77/78/79
