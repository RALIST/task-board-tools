# TB-36: TaskDrawer: inline metadata editing and Archive button

**Type:** feature
**Priority:** P1
**Size:** M
**Module:** gui
**Tags:** milestone-m3,frontend,drawer
**Branch:** —
**Parent:** TB-3

## Goal

Make priority, type, size, module, and tags editable in the drawer (Save → tb edit). Add Archive button (tb close).

## Acceptance Criteria

- [ ] Priority, type, size, module, tags become editable controls in `TaskDrawer.svelte`
- [ ] "Save" button is enabled only when fields are dirty; calls `BoardService.EditTask` with just the changed fields
- [ ] On success: toast confirmation + drawer reflects new state; on error: revert + toast
- [ ] "Archive" button at the drawer footer calls `BoardService.CloseTask`; closes drawer on success; card disappears from the active board within 1s (unless "Show archived" is on — handled by TB-38)
- [ ] Archive button requires confirmation (single-click confirm pattern: first click prompts "Click again to archive")
- [ ] Tags edited as a comma-separated input, trimmed and deduped before sending
- [ ] Depends on TB-32
