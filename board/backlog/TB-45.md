# TB-45: Agent state writer: JSONL events + per-run log file

**Type:** feature
**Priority:** P1
**Size:** M
**Module:** gui
**Tags:** milestone-m4,agent,state,jsonl
**Branch:** —
**Parent:** TB-4

## Goal

Append-only JSONL writer for run lifecycle events under `board/.agent-state/<ID>.jsonl`, plus a per-run log-file writer under `board/.agent-logs/<ID>/<run_id>.log`. Locks down the event vocabulary the rest of M4 uses.

## Acceptance Criteria

- [ ] `gui/internal/agent/state.go` exports `AppendEvent(boardDir, taskID string, event Event) error` — opens with `O_APPEND|O_CREATE|O_WRONLY`, writes one JSON object per line followed by `\n`, fsyncs and closes
- [ ] Event vocabulary is closed: `queued`, `started`, `stdout`, `stderr`, `finished`. No other JSONL event names are emitted anywhere in M4. (`stdout` and `stderr` are separate so the consumer can colour them differently.) The mapping to Wails IPC events is defined in TB-47
- [ ] Event shapes: `queued{ts, run_id, task_id, agent, mode}`, `started{ts, run_id, task_id, pid}`, `stdout{ts, run_id, task_id, line}`, `stderr{ts, run_id, task_id, line}`, `finished{ts, run_id, task_id, status, exit_code, reason?}`. `task_id` is on every event so `ListRuns` and a future log-trawler don't need cross-file lookups. `status ∈ {success, failed, cancelled}`
- [ ] `NewLogWriter(boardDir, taskID, runID) (io.WriteCloser, error)` returns a writer for `board/.agent-logs/<ID>/<run_id>.log`; creates the directory if missing; caller closes after the run
- [ ] `GenerateRunID() string` returns `r_<8 lowercase hex chars>` sourced from `crypto/rand`
- [ ] Concurrent-safe: lines never interleave. POSIX guarantees `O_APPEND` writes smaller than `PIPE_BUF` are atomic; document this guarantee in source and back it up with a per-task mutex as a belt-and-braces measure
- [ ] Tests: round-trip encode/decode for every event shape; concurrent goroutines appending 100 events each produce 100×N well-formed lines; rotation creates a new log file per `run_id`; missing-board-dir returns a typed error before any file is created
